fn needs_render(&self, n: &dyn Node) -> bool

Does the node need to render in the next sweep? This checks if the node is currently hidden, and if not, signals that we should render if the node is tainted, its focus status has changed, or if it is forcing a render.

fn is_on_focus_path(&self, n: &mut dyn Node) -> bool

Is the specified node on the focus path? A node is on the focus path if it has focus, or if it’s the ancestor of a node with focus.
