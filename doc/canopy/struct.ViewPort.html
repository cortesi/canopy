Struct canopy::ViewPort

source · 
pub struct ViewPort {
    pub projection: Point,
    pub view: Rect,
    pub size: Expanse,
}
Expand description

A ViewPort manages the size of a node and its projection onto the screen.

Fields§

§projection: Point

The location on screen that the view is projected to, specified as the co-ordinates of the top-left corner of the rectangle in the co-ordinate system of the parent node.

§view: Rect

A view within the size rectangle.

§size: Expanse

The total size of the node.
