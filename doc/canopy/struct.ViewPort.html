Expand description

A ViewPort manages the size of a node and its projection onto the screen. In many ways, this is the core of Canopy, and the viewport structure and its constraints determines many aspects of Canopyโs layout and rendering behavior.

A Canopy app is a tree of nested ViewPorts, with each maintaining a set of constraints with reference to its parent and children.
