Canopy: A terminal UI library.

Canopy is a terminal UI library for building interactive applications. It provides an arena-based widget system with focus management, styling, and event handling.

§Quick Start

The main entry points are:

  • Canopy - The core application state
  • Core - The arena and layout engine
  • Widget - The trait implemented by all widgets

§Module Organization

  • [geom] - Geometry primitives (Rect, Point, Expanse, etc.)
