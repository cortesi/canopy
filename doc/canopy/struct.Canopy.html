pub fn command_availability_from_focus(&self) -> Vec<CommandAvailability<'_>>

Return command availability from the current focus position.

This computes which commands would resolve to a target if dispatched from the current focus. For each command:

  • Free commands always have resolution = Some(Free)
  • Node-routed commands have resolution = Some(Subtree{..}) or Some(Ancestor{..}) if a matching node exists, None otherwise
Source

pub fn command_availability_from_node( &self, start: NodeId, ) -> Vec<CommandAvailability<'_>>

Return command availability from a specific node.

Computes which commands would dispatch to a target, using the same resolution logic as commands::dispatch:

  1. First search the subtree rooted at start in pre-order
  2. Then walk ancestors
