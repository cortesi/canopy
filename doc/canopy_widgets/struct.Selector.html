Struct Selector

pub struct Selector<T>
where
    T: SelectorItem,
{ /* private fields */ }
Expand description

A multi-select widget with checkbox-style items.

Items can be toggled on/off independently. The selected indices are tracked in the order they were selected, allowing for ordered selection if needed.
