Expand description

A position that can be clamped within the bounds of a State.

fn shift(&self, s: &State, n: isize) -> Self

Shift the cursor by an offset within a chunk. If the new position is out of bounds, return the closest matching position within the chunk. If the new offset lands on a character that is between lines, we continue in the same direction until we find a character that is in bounds.
